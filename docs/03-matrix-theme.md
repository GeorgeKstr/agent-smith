# Matrix Theme

## Boot screen

Shows:

- AGENT SMITH title
- falling green symbol lines
- sunglasses ASCII badge
- phase
- progress bar
- scanned files
- dirty files
- current file
- rotating tip

## Main screen

Panes:

- Project Intel
- Task Console
- Context Packet
- Patch / Test Output
- Status Bar

## Palette

- greenBright: primary title
- green: core UI
- gray: matrix rain / secondary text
- cyan: tips and model/context accents
- yellow: dirty/stale warnings
