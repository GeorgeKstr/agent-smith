# Agent Smith - Product Vision

Agent Smith is a Matrix-themed local-first terminal coding agent.

It helps local Ollama models work on medium codebases by keeping a persistent project intelligence index and feeding models small, dense context packets.

## Principle

The app is the agent. The model is a constrained component.

## The app handles

- indexing
- watching
- retrieval
- graph traversal
- context packing
- patch validation
- checks
- re-indexing

## The model handles

- task classification
- file/symbol summaries
- patch generation
- test failure interpretation

## Visual style

- green-on-black Matrix interface
- falling symbols / binary rain
- Agent Smith sunglasses badge
- sci-fi boot animation
- HUD panes
